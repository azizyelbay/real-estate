## Installing for Apache Kafka
Can be installed and run as:
### Docker
**1. Clone the application**

```bash
git clone https://github.com/azizyelbay/real-estate.git
```
**2. Go to the project directory**
```bash
cd real-estate
```
**3. Run**
```bash
docker compose up
```
